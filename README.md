# One Percent - Growth System App

一个把“成长系统”做成现实工具的 Android App。

如果你看过短剧、网文或小说里的“系统流”设定，这个项目可以理解为把那种体验落到日常生活里：把成长拆成明确维度，用记录、加分、扣分和趋势变化，让抽象的自我提升变得可见、可追踪、可积累。

## 亮点

- 用“系统”式的反馈感记录个人成长，不再只是写计划，而是把行动转成可累计的分数变化
- 把成长拆成一级维度和二级属性，既能看整体，也能看到具体是哪一项在提升或下滑
- 本地优先，离线可用，数据掌握在自己手里

## 主要功能

- 首页总览：查看总分、昨日变化和最近记录
- 维度管理：围绕多个一级成长维度组织个人成长内容
- 属性管理：新增、编辑、归档和恢复二级属性
- 记录打分：针对具体属性进行加分或扣分，形成成长轨迹
- 趋势分析：查看 7 天 / 30 天的分数变化趋势
- 数据管理：支持本地备份导出、导入恢复与清空数据

## 技术栈

- Flutter
- Dart
- SQLite (`sqflite`)
- `path_provider`
- `file_picker`
- `share_plus`

## 安装到手机

- 安装包路径：`build/app/outputs/flutter-apk/app-release.apk`
- 将 `app-release.apk` 发送到手机，可以用微信、QQ、数据线、网盘或局域网传输
- 在手机上打开该 `.apk` 文件，按提示安装
- 如果系统提示“禁止安装未知应用”，在手机设置里允许当前应用安装未知来源应用后再重试
